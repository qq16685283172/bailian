# 百炼计划成长中心 - 部署说明

## 一、文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主程序（包含用户端和管理后台） |
| `data.json` | 用户成长数据文件 |

## 二、部署方式（推荐：Gitee Pages）

### 第一步：注册 Gitee

1. 打开 [https://gitee.com](https://gitee.com)
2. 点击右上角「注册」，用手机号注册账号
3. 注册完成后登录

### 第二步：创建仓库

1. 点击右上角「+」→「新建仓库」
2. 仓库名称：`growth-center`（可任意取名）
3. 选择「公开」
4. 点击「创建」

### 第三步：上传文件

1. 在仓库页面，点击「上传文件」按钮
2. 把 `index.html` 和 `data.json` 拖到上传区域
3. 点击「提交」

### 第四步：开启 Pages 服务

1. 进入仓库 →「服务」→「Gitee Pages」
2. 部署分支选择 `master`
3. 点击「启动」
4. 等待 1-2 分钟
5. 你会得到一个网址，例如：`https://你的名字.gitee.io/growth-center`

### 第五步：访问

1. 在浏览器打开上面获得的网址
2. 输入用户的精网号（例如 JH0001）即可查看成长档案

## 三、日常更新数据

运营每天完成长值统计后，按以下步骤更新：

1. 在浏览器打开你的网址，在网址后面加上 `?admin=1`（或者点击页面右上角的 ⚙ 图标）
2. 进入管理后台
3. 按格式粘贴CSV数据：

```
JH0001,张先生,45,2026-06-23,易学久会签到,1
JH0001,张先生,45,2026-06-23,易学久会观看≥20分钟,1
JH0001,张先生,45,2026-06-23,易学久会课堂互动,1
JH0002,李阿姨,18,2026-06-23,财经壹眼签到,1
```

说明：
- 每行一条记录，用英文逗号分隔
- 格式：精网号,昵称,成长值,日期,来源,点数
- 如果同一个人有多个来源，写多行

4. 点击「导入预览」查看数据是否正确
5. 点击「下载 data.json」将更新的数据下载到电脑
6. 登录 Gitee，进入仓库，删除旧的 `data.json`，上传新的 `data.json`
7. 等待 1-2 分钟，用户刷新页面即可看到最新数据

## 四、CSV 数据格式说明

运营从 Excel/统计系统中导出的数据，整理成以下格式：

```
精网号,昵称,成长值,日期,来源,点数
JH0001,张先生,45,2026-06-23,易学久会签到,1
```

成长来源参考值：
- 易学久会签到
- 易学久会观看≥20分钟
- 易学久会课堂互动
- 财经壹眼签到
- 财经壹眼观看≥20分钟
- 财经壹眼课堂互动

## 五、没有网络的情况（纯本地使用）

如果你暂时不想部署到网上，也可以直接双击打开 `index.html` 使用：

1. 用 Chrome 浏览器打开 `index.html`
2. 数据会从 `data.json` 读取
3. 管理后台的 CSV 导入和下载功能仍然可用
4. 缺点：每次更新数据后，需要把新的 `data.json` 发给用户

## 六、常见问题

**Q: 用户输入精网号后显示「未找到」**
A: 检查 `data.json` 中是否包含该精网号，注意大小写。

**Q: 管理后台导入后排行榜不对**
A: 系统会自动按周/月/累计计算排行，导入时需包含足够的日期数据。

**Q: 页面显示乱码**
A: 确认文件是 UTF-8 编码。用记事本打开 → 另存为 → 编码选择 UTF-8。
